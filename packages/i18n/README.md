# @yaebal/i18n

a plural form set keyed by `Intl.PluralRules` categories. `other` is required
as the fallback; the rest are optional and selected per locale via
`new Intl.PluralRules(locale).select(n)`.

## install

```sh
pnpm add @yaebal/i18n
```
